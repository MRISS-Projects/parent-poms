Git

Setup

  1. Git should be installed and git executable should be in the path. In windows based systems the path should usually contains [GIT_HOME]\bin.
  2. Set environment variable GIT to true.
    1. In linux: 
      export GIT=true
    2. In windows: 
      set GIT=true

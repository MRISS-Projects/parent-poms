Proxy Configuration

Generic Proxy Server

Windows

Linux

Using linux 14.04, press at the "System Settings" button at the main applications bar.

Select "Network".

Select "Network Proxy". Set Method to Manual, and fill the HTTP and HTTPS proxy information with the address mentioned above. Click "Apply system wide".

Inside the browsers, then you can set them to use the global system settings. You can ignore proxy settings for localhost.
